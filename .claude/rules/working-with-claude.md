# Working with Claude in this project

Lightweight, optional conventions for building this app with Claude Code.
Delete this file if you don't want them — nothing depends on it.

- **Keep `handoff.md` current.** It holds where things stand plus the next step.
  Update it when you stop mid-task so the next session (you or the agent) can
  resume without re-deriving context.
- **Keep a running `journal.md`.** Append notable decisions newest-first, with
  the *why*. It's a log, not dated volumes — keep it simple.
- **Preserve, don't destroy.** Archive superseded code to
  `.archive/<slug>-<date>/` rather than deleting it; read a file before
  overwriting it.
- **Read `.claude/rules/authoring-panels.md` before building or editing a
  panel** — it's the chrome contract the kit is built around.
